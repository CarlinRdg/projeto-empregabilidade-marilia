# Conclusão

O presente projeto teve como objetivo analisar indicadores de empregabilidade utilizando dados públicos do Novo CAGED, com foco no município de Marília-SP.

A solução desenvolvida permitiu integrar diferentes tecnologias amplamente utilizadas em projetos de dados, incluindo PostgreSQL, Python, SQL e Power BI, possibilitando a coleta, tratamento, armazenamento e análise das informações.

Os resultados demonstraram que o município apresentou saldo positivo de empregos formais no período analisado, indicando desempenho favorável do mercado de trabalho local.

Além dos resultados obtidos, o projeto reforça a importância da utilização de dados públicos para apoiar decisões baseadas em evidências, contribuindo para o desenvolvimento regional e para a democratização do acesso à informação.

## Limitações

O estudo utilizou dados referentes à competência de setembro de 2019, limitando a análise temporal e impossibilitando a identificação de tendências de longo prazo.

Além disso, os dados utilizados são agregados, não permitindo análises detalhadas por perfil individual dos trabalhadores.

## Trabalhos Futuros

Como evolução deste projeto, recomenda-se:

* Ampliação da base de dados para múltiplos anos.
* Inclusão de indicadores educacionais e socioeconômicos do IBGE.
* Desenvolvimento de dashboards mais completos no Power BI.
* Implementação de análises preditivas utilizando técnicas de Ciência de Dados e Machine Learning.
