# Dashboard

Foi realizada a conexão entre o Power BI e o banco PostgreSQL desenvolvido no projeto.

As tabelas criadas no banco foram importadas com sucesso para o Power BI, permitindo a construção de indicadores e visualizações.

Como prova de conceito foram implementados cartões contendo:

* Total de admissões
* Total de desligamentos
* Saldo de empregos
* Variação percentual

A estrutura criada permite futuras expansões com gráficos comparativos por município, setor e subsetor econômico.
